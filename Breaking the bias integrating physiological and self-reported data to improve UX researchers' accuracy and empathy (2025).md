# Breaking the bias: integrating physiological and self-reported data to improve UX researchers' accuracy and empathy
# 詳細解説（2025）

---

# 1. 論文概要

## タイトル
Breaking the bias: integrating physiological and self-reported data to improve UX researchers' accuracy and empathy

## 公開年
2025

## 分野
- User Experience (UX)
- Physiological Computing
- Human-Computer Interaction (HCI)
- Emotion Recognition
- Affective Computing
- UX Research Methodology

---

# 2. この論文は何を目指したのか？

この研究の目的は、

> 「UX研究者の主観バイアスを減らし、より正確で共感的なUX評価を実現する」

ことである。

そのために：

- 生理データ（physiological data）
- 自己申告データ（self-reported data）

を統合した。

---

# 3. 背景：UX研究の大問題

従来のUX研究は：

- アンケート
- インタビュー
- Think Aloud
- SUS
- UEQ

など、

# 「ユーザーが言ったこと」

に強く依存している。

しかし人間には問題がある。

---

# 4. 人間の主観評価の限界

ユーザーは：

- 感情を忘れる
- 正確に言語化できない
- 社会的に望ましい回答をする
- 自分でも気づいていない

場合が多い。

例：

```text
本当はストレス
↓
でも「大丈夫でした」と回答
```

これが：

# Self-report bias（自己申告バイアス）

である。

---

# 5. UX研究者側にもバイアスがある

論文で特に重要なのがここ。

研究者も：

- 思い込み
- 先入観
- 仮説バイアス
- 確証バイアス

を持つ。

つまり：

```text
「このUIは使いやすいはず」
↓
都合よく解釈
```

してしまう可能性がある。

---

# 6. この研究の核心

この論文の核心は：

# 生理信号を「客観証拠」として使う

ことである。

---

# 7. 統合されるデータ

## 主観データ

- アンケート
- インタビュー
- 自己評価
- 感情レポート

---

## 客観データ

- 心拍（HR）
- 心拍変動（HRV）
- EDA（皮膚電気活動）
- 呼吸
- 視線
- 顔表情

---

# 8. なぜ生理信号が重要？

生理信号は：

> 「本人が気づいていない状態」

を反映できる。

例：

```text
本人：
「困ってないです」
```

でも：

- HRV低下
- EDA上昇
- 瞬目増加

なら：

# 実際はストレス状態

かもしれない。

---

# 9. UX評価の変化

従来：

# UX = 聞く

この研究：

# UX = 聞く + 測る

へ進化。

---

# 10. 実験構成

研究では：

- UX研究者
- ユーザー
- 生理データ

を組み合わせて実験。

研究者は：

- インタビューのみ
- 生理データ併用

でUX解釈を比較した。

---

# 11. 一番重要な結果

## 生理データ統合で解釈精度向上

研究者は：

- 生理データ無し
よりも、
- 生理データ有り

の方が：

# ユーザー状態を正確に理解

できた。

---

# 12. Empathy（共感）の改善

論文タイトルにもある重要概念。

## Empathyとは？

UX研究において：

> 「ユーザー体験を深く理解する能力」

のこと。

---

# 13. なぜ共感が改善した？

研究者は：

```text
「問題なさそう」
```

と思っていても、

生理データを見ると：

- ストレス
- 緊張
- 認知負荷

が発生していた。

これにより：

# 「ユーザーは実は困っていた」

と理解できた。

---

# 14. 重要概念：Hidden UX

この研究で重要なのは：

# Hidden UX

つまり：

> 表面化していないUX

である。

---

# 15. Hidden UXの例

```text
ユーザー：
「使えました」
```

しかし内部では：

- 迷い
- ストレス
- 認知負荷
- 微小フラストレーション

が存在。

生理データはこれを可視化する。

---

# 16. 使用される代表的生理指標

| 指標 | 意味 |
|---|---|
| HR | 緊張 |
| HRV | 自律神経 |
| EDA | ストレス |
| Blink | 疲労 |
| Pupil Diameter | 認知負荷 |
| Respiration | リラックス |
| Eye Tracking | 注意 |

---

# 17. HRV（Heart Rate Variability）

論文でも重要な指標。

## HRVとは

心拍間隔の揺らぎ。

### 高ストレス
HRV低下

### リラックス
HRV増加

つまり：

# UXストレス指標

として利用可能。

---

# 18. EDA（Electrodermal Activity）

EDAは：

# 発汗由来の皮膚電気変化

を計測。

特に：

- 緊張
- 驚き
- ストレス

に敏感。

---

# 19. 認知負荷（Cognitive Load）

この論文では：

# Cognitive Load

も重要。

---

## 認知負荷とは？

ユーザーが：

- 考え込む
- 理解に苦労する
- 探索する

ときに増える脳負荷。

例：

```text
「どこ押せばいい？」
```

状態。

---

# 20. UX研究へのインパクト

この論文は：

# UX研究をマルチモーダル化

した。

つまり：

- 主観
- 行動
- 生理

を統合する方向。

# 27. この研究の限界

論文では：

| 限界 | 内容 |

# 30. まとめ

> 「UX研究者の解釈バイアスを、生理データで補正する」

- HRV
- EDA
- 呼吸
