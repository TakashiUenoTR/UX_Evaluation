# Akan & Berkman (2020): ゲームユーザー研究における生理指標

## 書誌情報
- **タイトル**: Physiological Measures in Game User Research
- **著者**: Ekin Akan, Mehmet Ilker Berkman
- **出典**: *Game User Experience and Player-Centered Design*（書籍章）, Springer, 2020
- **DOI/URL**: https://link.springer.com/chapter/10.1007/978-3-030-37643-7_10
- **被引用数**: 約6（2026年時点）

## 概要
ゲームのユーザー研究（Game User Research, GUR）において、プレイヤーの体験（プレイヤー体験＝Player Experience, PX）を**生理指標**でどう評価するかを解説した書籍章。ゲームという強い情動を喚起する対象に対し、心拍以外の指標を含む多様な生理計測を整理する。

## 扱われている生理指標（心拍以外を中心に）
- **EDA（皮膚電気活動）**: 対戦相手が人間かAIかによってEDA反応が異なるなど、**情動的覚醒の計測**に利用。
- **表情筋EMG（Facial EMG）**: EDAと合わせ、**「対人対戦の方が情動が強い」ことをEMG結果が裏付けた**と報告。表情筋による情動価の高感度計測。
- **眼球運動（Eye tracking）**: 視覚的注意の配分を評価。
- **瞳孔測定（Pupillometry）**: 眼球運動に加え、瞳孔径から認知負荷・覚醒を評価。

## 主な知見
- **人間相手の対戦**では、AI相手より情動反応が強くなり、それが**EDAと表情筋EMGの両方で確認**された。
- 複数の生理指標を組み合わせることで、プレイヤーの情動体験を多面的に捉えられることを示す。

## 内容のポイント
1. **PX（プレイヤー体験）への特化**: ゲーム特有の没入・興奮・フラストレーションといった体験を生理指標で捉える。
2. **マルチモーダル計測**: EDA・EMG・眼球運動・瞳孔を組み合わせる実践的枠組み。
3. **社会的文脈の影響**: 対戦相手の性質（人間/AI）が情動反応に与える影響を実証的に示す。

## UX評価への示唆
- **ゲーム・エンタメUXの情動評価**に、EDA・表情筋EMGが特に有効であることを示す。
- 強い情動を喚起する体験では、複数の非心拍指標が一致した結果を示しやすい。
- 社会的・競争的文脈がUX（情動）に与える影響を生理指標で可視化できる。

## 引用の際のポイント
