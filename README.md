### 6. UX Estimation in HRI

**ファイル：** `User Experience Estimation in Human-Robot Interaction Via Multi-Instance Learning of Multimodal Social Signals (2025).md`

| 項目 | 内容 |
|---|---|
| タイトル | User Experience Estimation in Human-Robot Interaction via Multi-Instance Learning of Multimodal Social Signals |
| 著者 | Miyoshi, Okafuji, Iwamoto (CyberAgent), Nakanishi (Osaka Univ.), Baba |
| 掲載先 | IROS 2025 |
| arXiv | 2507.23544 |

#### 概要

人とロボットのインタラクション（HRI）において、顔表情と音声を用いたMulti-Instance Learningで UXを自動推定するTransformerベースモデルを提案。

#### モデル構成

```
顔画像 → Vision Encoder（2段階MIL + Transformer）  ─┐
                                                      → 特徴融合 → MLP → UXスコア（1〜7）
音声   → Audio Encoder（CNN + Transformer）          ─┘
```

#### 主な結果

| 評価方法 | Acc.7 | Acc.3 |
|---|---|---|
| チャンスレート | 14.3% | 33.3% |
| 提案手法（マルチモーダル） | 35.1% | 70%以上 |
| 人間の第三者評価者 | 約23% | 約64% |

**→ 提案手法が人間の第三者評価者を上回ることを実証**

#### キーワード

`HRI` `UX Estimation` `Multi-Instance Learning` `Multimodal` `Social Signals` `Transformer` `Facial Expression` `Voice`

### 7. User Experience Index (UXI)

**ファイル：** `User Experience Index (UXI).md`

| 項目 | 内容 |
|---|---|
| 種別 | 独自設計資料（論文ではない） |
| テーマ | UXIの概念・設計・計算方法 |

#### 概要

複数のUX指標（アンケート・生理データ・行動データなど）を統合し、UX価値を**単一スコア**で表現するための評価手法の設計資料。

#### UXIの目的

従来の多次元UX評価（Attractiveness, Efficiency, Novelty…）を集約し、

> 「この製品のUXは良いのか？」を直感的に判断できるスコアを提供する

#### 想定される構成要素

- UEQなどのアンケートスコア
- 生理信号（HRV・EDA・呼吸・瞬目など）
- 行動データ（視線・姿勢・ジェスチャ）
- AI感情推定結果

#### キーワード

`UXI` `UX Score` `Index` `Multimodal` `Aggregation` `Automotive UX`
